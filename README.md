# QuickCommands — Быстрые команды

Модуль для Topomatic Robur. Запуск команд Robur короткими командами (alias): через палитру с поиском и через опциональные нативные alias-команды.

## Возможности

- Палитра `qc` — поиск и запуск быстрых команд у курсора, без перезапуска Robur
- Нативные alias-команды — псевдоним работает как реальная команда Robur (набрал + Enter)
- Категории, импорт/экспорт наборов команд
- Защита от конфликтов имён с уже существующими командами Robur
- Подменю «Быстрые команды» в родном контекстном меню (ПКМ)

## Совместимость

| Продукт | Поддержка |
|---------|-----------|
| Топоматик Robur — Автомобильные дороги | ✓ |
| Топоматик Robur — Изыскания | ✓ |
| Топоматик Robur — Железные дороги | ✓ |
| Топоматик Robur — Искусственные сооружения | ✓ |
| Топоматик Robur — Инженерные сети | ✓ |
| Топоматик Robur — Генеральные планы | ✓ |

**Минимальная версия Robur:** 16.0.62.12

## Установка

Через менеджер модулей **ABR \| Библиотека модулей** (требуется модуль [AbrModules](https://github.com/Y-Abramov/AbrModules)):

1. Открыть **ABR → Модули → Управление модулями**
2. Найти **QuickCommands** в каталоге
3. Нажать **Установить** и перезапустить Robur
